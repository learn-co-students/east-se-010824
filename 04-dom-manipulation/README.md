# DOM Manipulation

### Learning Goals

&emsp; ✅ Explore how to traverse the DOM tree

&emsp; ✅ Create, read, update, and delete elements from the DOM by using: 
- `.querySelector()`
- `.querySelectorAll()`
- `.createElement()`
- `.getElementById()`
- `.getElementsByClassName()`
- `.append()`
- `.remove()`
- `.innerHTML` vs. `.innerText` vs. `.textContent`
